# Interpretable-Phishing-Detection-via-TF-IDF-n-grams-and-Compact-URL-Header-Features

This project focuses on **phishing e-mail detection** using **TF-IDF n-grams** and **compact URL/header features** to build interpretable models.  

## Datasets Used

The following datasets have been used in this project (files are **not included** in the repository):

- **TREC_07**: 2007 TREC phishing e-mail dataset  
- **CEAS_08**: 2008 CEAS phishing e-mail dataset  

> Please download these datasets from their official sources.
